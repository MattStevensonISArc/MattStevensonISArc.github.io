This page was created to demonstrate a series of JavaScript functions I have implemented.


If you enter some text into the TextArea and click the 'createLog()' button below, you will be asked to name your log, then the Date and Time will be automatically generated.




If you enter your own name and custom password into this function, it will not work. It has been set to only allow Username - 'Matt', and Password - 'Password01' to be accepted. If you use Inspect Element and direct to the Console you can see this function in action.
